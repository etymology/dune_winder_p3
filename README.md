# Chicago Winder GUI

## Description

This is the Chicago branch of the winder code ported to python3 (from 2.7) with modifications for our winding setup

## Installation

To run:

```bash
git clone ---------
cd dune_winder
pip install -r requirements.txt
source .venv/Scripts/activate
cd src
python main.py
```

## Maintenance

I use ruff to format. Run

```bash
pythom -m ruff format
pythom -m ruff check
```
